Tocá cuadrados del mismo color para encadenarlos, sin importar dónde estén. El eslabón gris deja cambiar a otro color sin cortar la cadena. Tocá uno ya marcado para explotar.

Cadena.

Los cuadrados caen y se amontonan. Marcá una cadena del mismo color (no importa la distancia entre ellos), usá el eslabón para saltar a otro color, y tocá de nuevo para hacerlos explotar.
